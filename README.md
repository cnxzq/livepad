# livepad

实时协作便签 — 所有人共享同一个 textarea，任意修改即时同步。

## 使用

```bash
npx livepad        # 默认 3000 端口
npx livepad 8080   # 指定端口
```

打开浏览器访问 `http://localhost:3000`，多个标签页/设备访问同一地址，在一个窗口中输入，所有窗口实时同步。

## 技术栈

- **Express** — HTTP 服务
- **Socket.IO** — 实时双向通信
- 纯原生 HTML/CSS/JS，零前端构建

## 原理

- 服务端内存中保存唯一一份共享内容
- 新客户端连接时接收当前内容
- 任意客户端输入 → 发送到服务端 → 广播给所有其他客户端
- 正在输入的客户端不回显自己的广播，避免光标跳动
